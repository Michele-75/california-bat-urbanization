# data/

This directory contains both raw source data (downloaded from external providers) and processed outputs generated by the preprocessing scripts. Raw data files are large and **not tracked by Git**. Processed files needed by the analysis notebook are included in the repository.

## Directory structure

```         
data/
├── raw/                          # (gitignored) Source data, see acquisition instructions below
│   ├── gbif/                     # GBIF bat occurrence downloads
│   ├── viirs/
│   │   ├── v21/                  # VIIRS VNL v2.1 annual GeoTIFFs (2012–2021)
│   │   └── v22/                  # VIIRS VNL v2.2 annual GeoTIFFs (2022–2024)
│   ├── boundaries/               # (auto-generated by tigris)
│   └── covariates/               # NLCD, PAD-US, GPWv4 rasters
├── processed/                    # Pipeline outputs
│   ├── gbif/                     # Cleaned bat points + presence panel
│   ├── boundaries/               # CA state boundary (.gpkg)
│   ├── grid/                     # 10 km analysis grid (.gpkg)
│   ├── covariates_grid/          # Grid-level covariate summaries
│   └── analysis_grid/            # Final modeling dataset
├── README.md                     # This file
└── data_dictionary.md            # Variable descriptions for final dataset
```

## Raw data acquisition

To reproduce the full pipeline from scratch, download the following datasets and place them in the indicated directories. All raw files are gitignored.

### 1. GBIF bat occurrences

-   Downloaded automatically by `preprocessing_scripts/01_get_clean_bat_points.R` using the `rgbif` package.

-   Requires GBIF credentials set in `.Renviron`:

    ```         
    GBIF_USER=your_username
    GBIF_PWD=your_password
    GBIF_EMAIL=your_email
    ```

-   Register for a free account at [gbif.org](https://www.gbif.org/).

### 2. VIIRS nighttime lights

-   **Source:** Earth Observation Group (EOG), Payne Institute, Colorado School of Mines
-   **URL:** <https://eogdata.mines.edu/products/vnl/>
-   **Product:** Annual VNL composites (average masked radiance), v2.1 for 2012–2021 and v2.2 for 2022–2024
-   **Placement:** `data/raw/viirs/v21/` and `data/raw/viirs/v22/` (one `.tif` per year)

### 3. NLCD land cover

-   **Source:** Multi-Resolution Land Characteristics Consortium (MRLC)
-   **URL:** <https://www.mrlc.gov/>
-   **Product:** Annual NLCD Land Cover 2019, CONUS (30 m)
-   **File:** `Annual_NLCD_LndCov_2019_CU_C1V1.tif`
-   **Placement:** `data/raw/covariates/`

### 4. PAD-US protected areas

-   **Source:** USGS Gap Analysis Project
-   **URL:** <https://www.usgs.gov/programs/gap-analysis-project/science/pad-us-data-download>
-   **Product:** PAD-US 4.1 Vector Analysis (PADUS Only), File Geodatabase
-   **File:** `PADUS4_1VectorAnalysis_PADUS_Only.gdb/` (directory)
-   **Placement:** `data/raw/covariates/`

### 5. Population density (GPWv4)

-   **Source:** NASA Socioeconomic Data and Applications Center (SEDAC), CIESIN, Columbia University
-   **URL:** <https://www.earthdata.nasa.gov/data/catalog/sedac-ciesin-sedac-gpwv4-popdens-r11-4.11>
-   **Product:** Gridded Population of the World, Version 4 (GPWv4): Population Density, Revision 11, 2020 estimate, \~1 km (30 arc-second) resolution
-   **File:** `population_density_2020.tif`
-   **Placement:** `data/raw/covariates/`

### 6. California state boundary

-   Downloaded automatically by `preprocessing_scripts/02_get_ca_boundary.R` using the `tigris` package.
-   No manual download required.

## Ethical considerations

-   **GBIF data governance:** All occurrence records are publicly shared under GBIF's data-use agreement. Individual observers are not identified in this analysis. The GBIF download DOI should be cited in any publication.
-   **Spatial precision:** Some records may have imprecise coordinates (`coordinate_uncertainty_in_meters` is retained for transparency). Records with obviously invalid coordinates were removed during cleaning.
-   **Observer bias:** GBIF records are opportunistic and spatially biased toward populated and accessible areas. This bias is acknowledged throughout the analysis and addressed by including population density as an accessibility proxy.
